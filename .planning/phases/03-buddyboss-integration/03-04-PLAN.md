How bp_core_load_template() works:
```php
// bp_core_load_template( 'members/single/plugins' ) locates the WordPress template
// file for the member profile plugin area. It fires the 'bp_template_content'
// action to render the actual content inside that wrapper.
//
// CRITICAL: Nothing is rendered inside the plugin area unless something is
// hooked onto 'bp_template_content' BEFORE bp_core_load_template() is called.
// The hook must be added inside the screen function itself — not at file scope.
//
// Correct pattern (from BuddyBoss core components e.g. bp-groups):
function bp_events_screen_attending() {
    add_action( 'bp_template_content', function() {
        bp_get_template_part( 'events/profile-attending' );
    } );
    bp_core_load_template( 'members/single/plugins' );
}
//
// Without the add_action call, bp_core_load_template fires but the content
// area is blank — the template part is never loaded.
```

    CRITICAL: The screen function MUST call add_action('bp_template_content', ...) BEFORE calling bp_core_load_template(). Without this, bp_core_load_template() fires but the content area renders blank — 'bp_template_content' is what actually injects the template part into the member profile plugin area.

    ```php
    <?php
    /**
     * BuddyBoss Events — Profile Attending Screen.
     *
     * Screen function for /members/{username}/events/attending.
     * Called by BP_Events_Component setup_nav registration.
     *
     * @package BuddyBoss\Events\Screens
     * @since BuddyBoss Events 1.0.0
     */
    defined( 'ABSPATH' ) || exit;

    /**
     * Set up the attending screen.
     *
     * Hooks the template part onto bp_template_content BEFORE loading the
     * member profile wrapper template. Without the add_action call, the wrapper
     * loads but the content area is blank.
     *
     * @since BuddyBoss Events 1.0.0
     */
    function bp_events_screen_attending() {
        /**
         * Fires before the attending screen template is loaded.
         *
         * @since BuddyBoss Events 1.0.0
         */
        do_action( 'bp_events_screen_attending' );

        add_action( 'bp_template_content', function() {
            bp_get_template_part( 'events/profile-attending' );
        } );

        bp_core_load_template( apply_filters( 'bp_events_template_attending', 'members/single/plugins' ) );
    }
    ```
