  <how-to-verify>
    Visit your MAMP BuddyBoss site at http://buddyboss-dev.local (or your configured URL).

    **1. Admin settings panel (ADMN-01, ADMN-02, ADMN-03)**
    - Log in as admin. Navigate to wp-admin → Settings → BuddyBoss → Events.
    - Confirm you see: "Who can create events" dropdown (Admins / Group Organizers / All Members), "Event moderation" toggle.
    - Navigate to wp-admin → Events → Revenue. Confirm event count stats appear (Total, Published, Pending, Draft).
    - Navigate to wp-admin → Events. Confirm the events list table shows a Pending tab.

    **2. Create an in-person event (EVNT-01, EVNT-04)**
    - Log in as a member with creation permission. Visit /events/create.
    - Complete the wizard: select In-Person, enter title "Test In-Person Event", set start/end dates, enter venue "City Hall, 123 Main St", capacity 50.
    - Click Save Draft. Confirm you are redirected to the event page and status shows Draft.
    - Edit the event and click Publish. Confirm the event appears on the /events/ calendar.

    **3. Create a virtual event (EVNT-02)**
    - Visit /events/create. Select Virtual. Enter a Zoom URL in the virtual URL field.
    - Publish directly. Confirm event appears on the calendar.

    **4. Create a recurring event (EVNT-03)**
    - Visit /events/create. Select In-Person. On the Date & Time step, check "Make this a recurring event".
    - Set weekly recurrence, every Monday, for 8 occurrences.
    - Publish. On the calendar, navigate forward — confirm 8 occurrences appear on Mondays.
    - Click on occurrence 3. Click Edit. Choose "Edit this event only". Change the title. Confirm occurrence 3 has the new title and others are unchanged.
    - Click on occurrence 5. Click Edit. Choose "Edit this and following". Change the title. Confirm occurrences 5+ have the new title; occurrences 3-4 are unchanged.

    **5. Calendar view toggle (locked decision)**
    - On the /events/ calendar, click the List toggle. Confirm the calendar switches to list view.
    - Click Month. Confirm it returns to month view.
    - Navigate to a different month using prev/next. Confirm events load without a full page reload.

    **6. Privacy enforcement (EVNT-05, EVNT-06)**
    - In wp-admin → Groups, find a private group. Create an event for that group.
    - Visit /events/ as a logged-out user or a non-member. Confirm the private group event does NOT appear on the site calendar.

    **7. Moderation flow (ADMN-02, ADMN-04)**
    - In admin settings, enable Event moderation.
    - Create a new event as a non-admin member. Confirm it lands in Pending (not Published on the calendar).
    - In wp-admin → Events → Pending, click Approve. Confirm the event is now published and visible on the calendar.
    - On a published event's page, confirm a "Report" link/button is visible. Click it and confirm the BuddyBoss moderation modal opens.

    **8. Permission enforcement (ADMN-01)**
    - In admin settings, set "Who can create events" to Admins only.
    - Log in as a regular member. Visit /events/create. Confirm access is denied or the Create button is hidden.
