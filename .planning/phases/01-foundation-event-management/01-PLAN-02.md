    Implement the following methods in class-bp-rest-events-endpoint.php:

    get_collection_params(): Return WP_REST_Controller::get_collection_params() merged with additional params:
    - 'from': ['type' => 'string', 'description' => 'Start date filter (MySQL or ISO8601)']
    - 'to': ['type' => 'string', 'description' => 'End date filter']
    - 'start': ['type' => 'string', 'description' => 'FullCalendar start (ISO8601)']
    - 'end': ['type' => 'string', 'description' => 'FullCalendar end (ISO8601)']
    - '_fc': ['type' => 'integer', 'description' => 'FullCalendar feed mode', 'default' => 0]
    - 'status': ['type' => 'string', 'default' => 'published']

    get_items_permissions_check($request): Return true for readable requests (public calendar); check is_user_logged_in() for write operations.

    get_items($request):
    - Map request params: $from = $request->get_param('start') ?: $request->get_param('from'); $to = $request->get_param('end') ?: $request->get_param('to');
    - Call bp_events_get_events(['from' => $from, 'to' => $to, 'per_page' => absint($request->get_param('per_page') ?: 200), 'status' => sanitize_key($request->get_param('status') ?: 'published')]).
    - Loop through results: foreach($events as $event) { $data[] = $this->prepare_item_for_response($event, $request)->get_data(); }
    - Return new WP_REST_Response($data, 200).

    prepare_item_for_response($event, $request):
    - Check $is_fc = (bool) $request->get_param('_fc').
    - If $is_fc is true, return rest_ensure_response(array('id' => $event->id, 'title' => $event->title, 'start' => str_replace(' ', 'T', $event->start_date), 'end' => str_replace(' ', 'T', $event->end_date), 'url' => bp_get_event_permalink($event), 'extendedProps' => array('type' => $event->type, 'venue' => ($event->venue_name ?: $event->virtual_url), 'status' => $event->status))).
    - If $is_fc is false, return the full BP_Event object fields as an associative array (all public properties).

    Follow WordPress Coding Standards throughout. Use absint(), sanitize_key(), esc_url_raw() where appropriate.
