    bp_events_split_series( $original_parent_id, $split_from_occurrence_id, $updated_fields ):

    - Delete all child rows with parent_event_id = $original_parent_id AND start_date >= split occurrence's start_date using $wpdb->delete() with prepare().

    - Return bool result.
