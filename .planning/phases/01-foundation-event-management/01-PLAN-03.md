    LEFT JOIN to the groups table:
    ```sql
    LEFT JOIN {$wpdb->prefix}bp_groups AS g ON e.group_id = g.id
    ```

    LEFT JOIN to groupmeta for the per-group site-calendar setting:
    ```sql
    LEFT JOIN {$wpdb->prefix}bp_groups_groupmeta AS gm
    ```

    Use $wpdb->prepare() for any user-supplied values in the query. Table names via $bp->groups->table_name and $wpdb->prefix . 'bp_groups_groupmeta'.

    Add a filter 'bp_events_get_events_where_clauses' so the WHERE array is filterable (follow BuddyBoss extensibility pattern).

    Do NOT modify the query when $r['group_id'] is explicitly set (group-scoped queries bypass site-wide privacy rules — the group controls its own view).
